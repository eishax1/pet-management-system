# Pet Management System

## Overview
The **Pet Management System** is a Python project that allows users to manage details about various pets. It provides functionalities to add new pets, display all pet details, search for specific pets, and visualize the composition of pets through graphs.

## Features
- Add new pets and store their details.
- Search for a pet by name.
- Display all pets' information.
- Identify the youngest and eldest pet.
- Sort pets by age.
- Display the number of pets managed by each owner.
- Generate a bar graph showing the composition of pets by type.
